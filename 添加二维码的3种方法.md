# 添加微信二维码的3种方法

## 方法1：使用图片文件（最简单）

1. 将你的微信二维码保存为图片
2. 重命名为 `wechat-qr.png`
3. 放在 `drinking-game/` 文件夹中
4. 完成！

**优点**：简单直接
**缺点**：需要管理图片文件

---

## 方法2：使用在线图片链接（推荐）

### 步骤：

1. **上传图片到图床**
   - 推荐图床：
     - 微信公众号后台（上传后复制图片链接）
     - 图床网站：https://imgbb.com/
     - GitHub：上传到仓库后获取链接
     - 七牛云、阿里云OSS等

2. **获取图片链接**
   - 上传后复制图片的完整URL
   - 例如：`https://example.com/my-wechat-qr.png`

3. **修改代码**
   - 打开 `drinking-game/index.html`
   - 找到这一行：
     ```html
     <img src="https://你的图片链接.com/wechat-qr.png" alt="微信二维码"
     ```
   - 替换成你的图片链接

**优点**：不需要管理本地文件，部署方便
**缺点**：需要图片托管服务

---

## 方法3：使用 Base64 编码（无需外部文件）

### 步骤：

1. **转换图片为 Base64**
   
   **在线工具**：
   - 访问：https://www.base64-image.de/
   - 或：https://base64.guru/converter/encode/image
   
   **操作**：
   - 上传你的二维码图片
   - 点击"Encode"或"转换"
   - 复制生成的 Base64 代码（以 `data:image/png;base64,` 开头的长字符串）

2. **修改代码**
   
   打开 `drinking-game/index.html`，找到二维码图片部分，替换为：
   
   ```html
   <img src="data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA..." 
        alt="微信二维码" 
        style="width:100%;max-width:240px;height:auto;display:block;margin:0 auto;">
   ```
   
   将 `data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA...` 替换成你的完整 Base64 代码

**优点**：
- 无需外部文件
- 无需图床服务
- 部署简单，一个 HTML 文件搞定

**缺点**：
- Base64 代码很长
- 文件体积会增大

---

## 推荐方案

### 如果你要部署到 GitHub Pages 或 Cloudflare Pages：
→ 使用**方法1**（图片文件）或**方法2**（在线链接）

### 如果你只想快速测试或分享单个 HTML 文件：
→ 使用**方法3**（Base64 编码）

## 快速操作指南

### 我现在就想添加二维码，怎么做最快？

**最快方法**：

1. 打开微信，截图你的二维码
2. 保存图片，重命名为 `wechat-qr.png`
3. 把图片拖到 `drinking-game/` 文件夹里
4. 打开 `index.html` 测试

**5秒搞定！** ✅

## 常见问题

### Q: 图片不显示怎么办？
A: 检查：
1. 文件名是否正确：`wechat-qr.png`（注意大小写）
2. 文件是否在正确位置：与 `index.html` 同一文件夹
3. 浏览器控制台是否有错误（按 F12 查看）

### Q: 可以用 JPG 格式吗？
A: 可以，但需要修改代码中的文件名：
```html
<img src="wechat-qr.jpg" alt="微信二维码">
```

### Q: 二维码太大或太小怎么办？
A: 修改 `max-width` 值：
```html
style="width:100%;max-width:240px;..."
```
改成 `max-width:300px` 或其他尺寸

### Q: 我想用微信号代替二维码可以吗？
A: 当然可以！修改弹窗内容：
```html
<div style="background:#FFF;padding:20px;border-radius:12px;margin-bottom:16px;">
  <p style="font-size:18px;font-weight:700;color:var(--text-red);">
    微信号：your_wechat_id
  </p>
  <p style="font-size:12px;color:var(--text-muted);margin-top:8px;">
    添加时请备注：酒桌模拟器
  </p>
</div>
```
