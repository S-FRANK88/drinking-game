## 🧪 测试步骤

### 方法1：直接打开 HTML 文件

1. 找到 `drinking-game/index.html`
2. 双击打开（会在浏览器中打开）
3. 点击右下角的 💬 按钮
4. 查看二维码是否正常显示

### 方法2：使用本地服务器（推荐）

如果直接打开看不到图片，可能是浏览器安全限制，需要用本地服务器：

**使用 Python（如果已安装）：**
```bash
cd drinking-game
python -m http.server 8000
```
然后访问：http://localhost:8000

**使用 Node.js（如果已安装）：**
```bash
cd drinking-game
npx http-server -p 8000
```
然后访问：http://localhost:8000

**使用 VS Code：**
- 安装 "Live Server" 插件
- 右键 `index.html` → "Open with Live Server"

## 🔍 检查清单

如果二维码不显示，请检查：

- [ ] 文件名是否正确：`wechat-qr.png`（注意大小写）
- [ ] 文件是否在正确位置：与 `index.html` 同一文件夹
- [ ] 图片格式是否正确：PNG、JPG 都可以
- [ ] 浏览器控制台是否有错误（按 F12 查看）
- [ ] 是否使用了本地服务器打开（而不是直接双击）
