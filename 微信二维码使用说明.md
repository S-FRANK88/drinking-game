# 微信二维码功能使用说明

## 功能说明

在游戏界面右下角添加了一个微信二维码浮动按钮（💬），点击后会弹出二维码弹窗，方便玩家联系作者。

## 使用步骤

### 1. 准备二维码图片

1. 打开微信，进入"我" → "二维码名片"
2. 截图或保存你的微信二维码
3. 将图片命名为 `wechat-qr.png`
4. 将图片放在 `drinking-game/` 文件夹中（与 index.html 同级）

### 2. 图片要求

- 文件名：`wechat-qr.png`（必须是这个名字）
- 格式：PNG、JPG 都可以（但文件名必须是 .png）
- 尺寸：建议 400x400 像素以上，保证清晰度
- 位置：`drinking-game/wechat-qr.png`

### 3. 功能特点

- 浮动按钮位于右下角，金色背景，💬 图标
- 按钮有悬停动画效果
- 点击后弹出精美的二维码弹窗
- 弹窗包含：
  - 二维码图片
  - 个人简介："物理老师 · 戏剧人 · AI折腾家"
  - 欢迎语："欢迎交流游戏创意和AI应用"
- 如果图片未找到，会显示友好的提示信息

### 4. 自定义文案

如果你想修改弹窗中的文案，可以编辑 `index.html` 文件中的这部分：

```html
<p style="font-size:12px;color:var(--text-muted);line-height:1.6;margin-bottom:16px;">
  物理老师 · 戏剧人 · AI折腾家<br>
  欢迎交流游戏创意和AI应用
</p>
```

## 注意事项

1. 确保二维码图片清晰，方便扫描
2. 定期更新二维码，避免过期
3. 如果不想显示二维码按钮，可以在 index.html 中删除或隐藏相关代码
